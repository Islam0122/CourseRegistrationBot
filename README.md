# CourseRegistrationBot

Telegram-бот для регистрации студентов на курсы с использованием **Telegram Bot API**, фреймворка **Aiogram**, базы данных SQLite и внешних API/Web Scraping.

---

## 🎯 Цель проекта

Разработать Telegram-бота, который позволяет:

- Регистрировать студентов на курсы
- Просматривать доступные курсы
- Отслеживать количество зарегистрированных студентов
- Управлять регистрацией (добавление, удаление)
- Администрировать курсы
- Автоматически получать и обновлять список курсов с внешнего источника

Проект реализуется поэтапно с применением FSM, базы данных, кнопок, middleware и деплоя.

---

## 📌 Функциональные требования

### 1. Регистрация пользователя

- Автоматическая регистрация при первом запуске
- Сохранение user_id, имени и даты регистрации в БД

### 2. Просмотр курсов

- Список доступных курсов
- Возможность выбрать курс для регистрации
- Inline кнопки для выбора курса

### 3. Регистрация на курс

- FSM шаги:
  1. Выбор курса
  2. Подтверждение регистрации
- После регистрации выводится сообщение:

```text
Вы успешно зарегистрированы на курс <название курса> ✅
````

### 4. Отмена регистрации

* Пользователь может отменить регистрацию на курс
* Inline кнопка ❌ Отменить
* Проверка, что студент не зарегистрирован дважды на один курс

### 5. Админ-панель

* Доступ только для admin_id
* Просмотр всех студентов
* Просмотр студентов по курсу
* Добавление/удаление курсов
* Обновление списка курсов через `/update_courses`

### 6. Интеграция API / Web Scraping

* Получение актуальных курсов с внешнего API или сайта
* Обновление локальной таблицы `courses`
* Обработка ошибок соединения / недоступности
* Команда `/courses` для отображения актуальных курсов

### 7. Middleware

* Логирование действий пользователя
* Проверка регистрации пользователя
* Ограничение доступа к админ-командам

### 8. Деплой

* Бот должен работать 24/7 на Heroku или VPS
* Автозапуск после перезагрузки
* Логи сохраняются на сервере
